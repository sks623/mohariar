===============================================================================

- openpyxl (Excel read/write)
- PyPDF2 (PDF merging)
- pyautogui (automation)
- comtypes (Office automation)
- win10toast (notifications)

===============================================================================
TROUBLESHOOTING
===============================================================================

Problem 1: "Python is not recognized"
--------------------------------------
Solution:
- Reinstall Python and check "Add Python to PATH"
- OR manually add Python to PATH:
  1. Search "Environment Variables" in Windows
  2. Edit "Path" variable
  3. Add: C:\Users\YourName\AppData\Local\Programs\Python\Python3X\


Problem 2: "pip is not recognized"
-----------------------------------
Solution:

--- COPY FROM BELOW ---

python -m pip install -r requirements.txt

--- COPY UNTIL ABOVE ---


Problem 3: Chrome Driver Error
-------------------------------
Solution:
- Make sure Google Chrome browser is installed
- ChromeDriver is automatically downloaded on first run
- If still having issues, check your internet connection
- The app needs internet to download ChromeDriver on first run


Problem 4: Module Not Found Error
----------------------------------
Solution:
- Install missing module manually:

--- COPY FROM BELOW ---

pip install customtkinter selenium pandas openpyxl PyPDF2 pyautogui comtypes win10toast pillow webdriver-manager

--- COPY UNTIL ABOVE ---


Problem 5: "SSL Certificate Error"
-----------------------------------
Solution:

--- COPY FROM BELOW ---

pip install --trusted-host pypi.org --trusted-host files.pythonhosted.org -r requirements.txt

--- COPY UNTIL ABOVE ---
